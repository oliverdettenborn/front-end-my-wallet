### Features
- [x] Registration and login<br>
- [x] Initial screen listing all posted entry and exit transactions<br>
- [x] Current balance display<br>
- [x] Button for adding a new entry<br>
- [x] Button for adding new output<br>
- [ ] Field for the user to specify the date of entry or exit<br>
- [ ] When clicking on an entry / exit, a screen for editing the entry / exit or deleting it is displayed<br>
- [ ] Searched period filtering: "All" / "7 days" / "15 days" / "30 days"<br>
- [ ] Category management for inputs or outputs<br>
- [ ] Category filtering<br>
